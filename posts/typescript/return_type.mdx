Here's a simple example of how to use `ReturnType`:

```ts
function getMessage() {
  return "Hello, TypeScript!";
}

type MessageType = ReturnType<typeof getMessage>;
// MessageType is 'string'
```

easy! right?

so let imagine that you have a function which return a ton of different types

```ts
function get() {
  return "Hello, TypeScript!";
}
```
